### 第二层：3个投资经理辩论 ⭐ 核心创新

基于7个研究员的观点，3个不同风格的投资经理进行**激烈辩论**：

| 投资经理 | 风格 | 人格特点 | 决策依据 |
|:-------:|:-----|:---------|:--------|
| 🔥 **激进型PM** | 追求高收益 | 进攻性强，敢于质疑保守观点 | 主要参考舆情+技术面+资金流 |
| ⚖️ **中性型PM** | 平衡风险收益 | 理性客观，调和两方观点 | 综合7个研究员观点 |
| 🛡️ **保守型PM** | 优先保本 | 强调风险，质疑乐观预期 | 主要参考基本面+估值+国际宏观 |

#### 辩论机制设计

**辩论流程**：
```
1. 激进型PM先发言 → 阐述看多理由，质疑保守观点
2. 保守型PM反驳 → 指出风险，质疑激进观点
3. 中性型PM调和 → 综合两方观点，提出平衡方案
4. 风险管理委员会 → 综合三方观点，做出最终决议
```

**辩论风格示例**：

**激进型PM**：
> "你们说'持有'太保守，我说'做多'才对得起趋势！你们看到的是风险，我看到的是机会。8月6日放量上涨7.75%，这不是散户追高，这是机构抢筹！你们因为一个'可能'的回调就建议持有，是不是太早放弃上涨空间了？"

**保守型PM**：
> "你看到的到底是趋势，还是泡沫？顶背离往往是趋势衰竭的前兆。你说是机构布局，那为什么缩量回调时没有资金承接？你敢拿公司资金去赌这个趋势吗？保守不是错，而是责任。"

**中性型PM**：
> "你们一个太冲，一个太守。投资不是非黑即白。现在最需要的是'适度风险、动态调整'的策略。建议在275-280区间轻仓介入，设置止损260，止盈300。不是赌趋势，也不是怕风险，而是在风险和收益之间找到平衡。"

### 第四层：风险管理官（审核层）

**风险矩阵**：

| 风险类别 | 评估内容 | 风险等级 |
|:-------:|:--------|:-------:|
| 系统性风险 | 美股暴跌、地缘冲突 | 低/中/高 |
| 政策风险 | 监管收紧、政策转向 | 低/中/高 |
| 流动性风险 | 成交量萎缩、北向流出 | 低/中/高 |
| 估值风险 | 估值过高、泡沫化 | 低/中/高 |
| 外部风险 | 汇率波动、大宗商品 | 低/中/高 |

**强制规则检查**：
- [ ] 单票持仓2-10%
- [ ] 单行业不超过30%
- [ ] 止损纪律（-8%触发）
- [ ] 现金仓位符合市场情绪
- [ ] 投资范围限定（A股/港股/黄金）

**风控决策**：
- ✅ 批准：风险可控
- ⚠️ 修改建议：降低仓位/增加止损
- ❌ 否决：风险过高

### 信心指数计算

```
信心指数 = (研究员共识度 × 30% + 投资经理共识度 × 30% + 辩论质量 × 20% + 风控评级 × 20%) × 100

研究员共识度 = 一致观点数量 / 7
投资经理共识度 = 一致建议数量 / 3
辩论质量 = 论据充分性 + 逻辑严密性
```

## 辩论机制的价值

| 优势 | 说明 |
|------|------|
| **观点碰撞** | 激进与保守的对抗，暴露潜在风险和机会 |
| **避免偏见** | 单一观点容易偏颇，辩论机制强制考虑多方面 |
| **决策透明** | 用户可以看到完整的决策推理过程 |
| **人格化表达** | 生动的辩论风格，更易理解和记忆 |
| **分层建议** | 不同风险偏好的投资者都能找到适合的策略 |
