# Obsidian CLI Skill

Bundled copy of the canonical `obsidian-cli` skill for `claude-code-skills`.

This skill enables interaction with Obsidian vaults via the built-in CLI (v1.12.6+), including:

- file management
- full-text search
- tag and property management
- backlink and link graph queries
- plugin and theme management
- bases queries
- workspace and sync operations

## Requirements

- Obsidian `1.12.0+` with CLI enabled
- Catalyst license
- macOS or Linux
- Obsidian app running
