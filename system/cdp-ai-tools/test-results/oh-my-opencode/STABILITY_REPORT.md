1. **插件版本过时** ⚠️
   - 当前版本: 3.8.5
   - 最新版本: 3.9.0
   - 影响: 插件功能
   - 修复: `cd ~/.config/opencode && bun update oh-my-opencode`

2. **AST-Grep 不可用** ⚠️
   - 影响: ast_grep_search, ast_grep_replace 功能
   - 修复: 安装 @ast-grep/cli 或 @ast-grep/napi

3. **Comment checker 不可用** ⚠️
   - 影响: comment-checker hook
   - 修复: 安装 @code-yeongyu/comment-checker

4. **未检测到 LSP 服务器** ⚠️
   - 影响: LSP 诊断、重命名、引用功能
   - 修复: 安装至少一个 LSP 服务器
