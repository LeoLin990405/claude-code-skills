## 💡 实现原理

CDP (Chrome DevTools Protocol) 允许我们：

1. **执行任意 JavaScript**
   - 操作 DOM 元素
   - 调用页面函数
   - 访问页面数据

2. **模拟用户操作**
   - 点击按钮
   - 输入文本
   - 键盘事件
   - 鼠标事件

3. **监控页面状态**
   - 监听事件
   - 获取网络请求
   - 截图
   - 性能分析

4. **注入自定义代码**
   - 修改页面样式
   - 添加新功能
   - 自动化脚本

## 🔧 如何添加新功能

### 步骤 1：探索 DOM 结构

使用浏览器开发者工具或 inspect_dom.py：

```bash
cd ~/.local/bin/doubao-cli
python3 inspect_dom.py
```

### 步骤 2：编写 JavaScript 代码

在浏览器控制台测试：

```javascript
// 查找元素
let element = document.querySelector('[class*="target"]');

// 执行操作
element.click();

// 获取数据
let data = element.textContent;
```

### 步骤 3：集成到 Python

在 ExtendedCDP 类中添加新方法：

```python
async def new_feature(self):
    js = """
    (function() {
        // 你的 JavaScript 代码
        return { success: true, data: result };
    })()
    """

    result = await self.execute_js(js)
    return result.get("result", {}).get("result", {}).get("value", {})
```

### 步骤 4：添加命令行参数

在 advanced_demo.py 中：

```python
parser.add_argument('--new-feature', action='store_true',
                    help='使用新功能')

if args.new_feature:
    result = await cdp.new_feature()
    print(result)
```
